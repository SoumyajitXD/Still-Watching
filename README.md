# Still Watching

**Still Watching** is a Forge 1.20.1 horror survival modpack built around fog, sound, stalking entities, cursed exploration, deadly nights, and paranoia.

The Anomaly is watching. The night is not safe. Silence does not mean peace.

## Official Download

Download and play the modpack on CurseForge:

https://www.curseforge.com/minecraft/modpacks/still-watching

This GitHub repository is for project files, documentation, issue tracking, and release organization. CurseForge is the main place to install the modpack.

## Features

- Forge 1.20.1 horror survival
- Fog, darkness, and paranoia-focused atmosphere
- Stalking horror entities, including **The Anomaly**
- Dangerous structures, dungeons, and exploration
- Sound-based tension and jumpscare-heavy gameplay
- Multiplayer support

## Recommended Setup
