
STILL WATCHING
==============

Forge 1.20.1 Horror Survival Modpack

Still Watching is a Minecraft horror survival modpack built around fog, sound, stalking entities, cursed exploration, deadly nights, and constant paranoia.

The Anomaly is watching.
The night is not safe.
Silence does not mean peace.

Will you survive?


OFFICIAL MODPACK PAGE
=====================

Still Watching is officially available on CurseForge:

https://www.curseforge.com/minecraft/modpacks/still-watching

Please use the CurseForge page to download, install, and play the modpack.

This GitHub repository is mainly for project files, configuration tracking, documentation, issue reports, and development reference. It is not intended to replace the official CurseForge download.


ABOUT THE MODPACK
=================

Still Watching turns Minecraft into a hostile horror survival experience where comfort dies quickly.

This is not a cozy survival pack with a few scary mobs sprinkled on top. The pack is designed to make the world feel wrong. Fog hides movement. Darkness becomes dangerous. Sounds matter. Structures are risky. Mining is no longer peaceful. Something is always watching from just outside the torchlight.

The pack focuses on:

- Cinematic horror atmosphere
- Stalking horror entities
- Fog, darkness, and paranoia
- Sound-based tension
- Cursed exploration
- Dangerous structures and dungeons
- Survival over power fantasy
- Multiplayer horror survival


MAIN FEATURES
=============

CINEMATIC HORROR ATMOSPHERE
---------------------------

Still Watching uses horror-focused visuals, dynamic lighting, ambient sound, audio physics, screen effects, and dark environmental design to make Minecraft feel hostile.

Even when nothing is attacking you, the world should still feel unsafe.


FOG, DARKNESS, AND PARANOIA
---------------------------

Fog is not decoration.
Darkness is not empty.
Silence is not safety.

Expect distant noises, strange sightings, fake comfort, sudden encounters, and moments where you are not sure if the danger is real or already behind you.


STALKING HORROR ENTITIES
------------------------

The pack includes terrifying creatures that hunt, observe, ambush, mimic, and punish careless players.

Some enemies attack immediately.
Some wait.
Some watch first.

That is worse.


SOUND-BASED FEAR
================

Audio awareness matters.

Footsteps, echoes, whispers, growls, knocks, and distant noises can all signal danger. Playing without sound is a great way to get folded by something you never saw coming.

Headphones are strongly recommended.


CURSED WORLD GENERATION
=======================

The world includes darker terrain, eerie biomes, abandoned locations, twisted structures, and dangerous dungeons.

Exploration should feel like entering somewhere you were never supposed to find.


EXPLORATION WITH CONSEQUENCES
=============================

Loot is not free.
Structures are not safe.
Dungeons fight back.

Curiosity can reward you, but it can also drag you directly into regret.


SURVIVAL, NOT POWER FANTASY
===========================

Weapons and guns may exist, but they do not make you invincible.

Ammo can run out.
Panic ruins aim.
Strong gear does not save bad decisions.

Survival depends on awareness, preparation, movement, and knowing when to run.


THE ANOMALY
===========

The Anomaly is the signature threat of Still Watching.

It is something that should not exist, but does anyway. Reality feels unstable around it. Safety becomes temporary. The rules stop feeling reliable.

You do not simply fight The Anomaly.

You endure it.

SERVER NOTES
============

Still Watching supports multiplayer.

For server use:

- Use the official CurseForge/server files if available.
- Keep client and server versions matching.
- Use enough RAM for your player count.
- Test the server before inviting players.
- Do not install client-only mods on the server.
- Check logs when crashes happen instead of guessing like a caveman with a keyboard.

Recommended server RAM depends on player count, world size, and configuration.


PERFORMANCE TIPS
================

If you experience low FPS, crashes, or stutters:

- Allocate at least 5GB RAM.
- Lower render distance.
- Lower simulation distance.
- Reduce shader settings.
- Disable shaders if needed.
- Close heavy background programs.
- Update GPU drivers.
- Avoid adding random extra mods.
- Test in a clean instance before blaming the pack.

More RAM does not magically fix everything. Sometimes the settings are just trying to cook your PC alive.
