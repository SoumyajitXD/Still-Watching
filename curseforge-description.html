WHAT IS STILL WATCHING?

Still Watching is a cinematic Minecraft horror modpack designed to make survival feel dangerous again. This is not cozy survival with a few monsters sprinkled on top. This is a hostile world of fog-covered forests, abandoned structures, distorted sounds, brutal nights, and creatures that stalk, hunt, mimic, and wait.

Build carefully. Explore slowly. Listen closely. Something is always wrong.

WHY PLAY THIS PACK?

🌑 CINEMATIC HORROR ATMOSPHERE
Fog, lighting, sound physics, screen distortion, blood effects, and horror-focused visuals turn Minecraft into a world that feels unsafe before anything even attacks.

🔊 SOUND-LED SURVIVAL
Footsteps, knocks, whispers, growls, echoes, and distant movement matter. Headphones are strongly recommended.

👁️ STALKING ENTITIES
Threats observe, ambush, mimic, chase, punish careless movement, and wait for players to relax. Rude, but effective.

🌲 CURSED EXPLORATION
Darker terrain, hostile structures, eerie towns, and unforgiving dungeons make curiosity dangerous instead of free.

🌘 BRUTAL NIGHTS
Night is not a scheduled inconvenience. It is where mistakes compound and fake safety gets punished.

🔫 SURVIVAL, NOT POWER FANTASY
Weapons and gear help, but they do not make you invincible. Ammo runs out. Panic ruins aim. Awareness stays king.

RECOMMENDED SETUP

  • Minecraft: 1.20.1
  • Mod Loader: Forge
  • Java: Java 17 if your launcher asks
  • RAM: 5GB minimum, 6–8GB preferred
  • Audio: Headphones strongly recommended
  • Multiplayer: Supported — horror is worse when your friends start screaming too

INSTALLATION

  1. Install the CurseForge App.
  2. Open Minecraft inside CurseForge.
  3. Search for Still Watching.
  4. Confirm the project is by Soumyajit.
  5. Click Install.
  6. Allocate at least 5GB RAM.
  7. Launch the pack and use headphones.

Do not download Still Watching from random reupload sites. Malware loves impatient players. Do not hand it a chair.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Keep client-only visual, shader, UI, and performance mods out of dedicated servers when required.
  • Test world generation before inviting players.

A mismatched server pack is not mysterious horror. It is just bad admin work wearing a cheap mask.

NEED HELP?

Before reporting an issue, test on a clean install if possible and include:

  • Modpack version
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Reports like “it broke” are not bug reports. They are haunted postcards. Bring evidence.
