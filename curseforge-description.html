STILL WATCHING

Forge 1.20.1 horror survival built around fog, sound, stalking entities, and the rotten feeling that you were noticed first.

The fog is not empty. The silence is not safety.

Build carefully. Explore slowly. Listen before you move.

Official downloads belong on CurseForge. Avoid random reupload sites unless malware is somehow part of your horror experience.

Best played at night with headphones, low brightness, and just enough confidence to make bad decisions.

THE 10-SECOND PITCH

Still Watching is a focused Minecraft horror survival modpack where the world stops feeling neutral. Fog hides movement. Audio becomes evidence. Caves, forests, structures, and roads all feel like places you should not stay in for long.

This is not a kitchen-sink pack wearing a cheap Halloween mask. It is a curated survival nightmare about paranoia, hostile exploration, and learning that shelter is only a temporary argument.

WHAT MAKES IT DIFFERENT

Fog-first atmosphere
Visibility is part of the threat. The pack uses distance, darkness, and terrain to make ordinary travel feel exposed.		 Sound-led horror
Footsteps, knocks, whispers, echoes, growls, and distant movement are not decoration. Ignoring them is how you donate your inventory.
Stalking entities
Threats do not exist just to jumpscare you once and leave. They hunt, follow, ambush, punish noise, and wait for careless routes.		 Survival pressure
Gear matters, but it does not turn you into a superhero. Awareness, preparation, and escape routes matter more.

THE EXPERIENCE LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore forests, caves, structures, lonely paths, and cursed places for loot and answers.
  3. Listen before moving. Sound is often the first warning and the last mercy.
  4. React when stalking entities turn your plan into comedy.
  5. Return with better gear, worse nerves, and fewer lies about being safe.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Do not dump client-only visual, shader, UI, map, or rendering mods into a dedicated server.
  • Test world generation and joining before inviting players.
  • Voice/audio setup is worth testing before the first real session.

A mismatched server pack is not mysterious horror. It is bad admin work wearing a cheap mask.

BEFORE REPORTING ISSUES

Reproduce the issue on a clean install if possible. If it still happens, report it on GitHub with useful evidence, not a haunted postcard that says “it broke.”

  • Modpack version
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Report issues on GitHub or read the installation guide.
