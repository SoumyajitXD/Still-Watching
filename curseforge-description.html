WHAT IS STILL WATCHING?

Still Watching is not a kitchen-sink pack with a cheap monster mask taped to its face. It is a focused horror survival experience where the world feels hostile before anything touches you.

Forests vanish into mist. Structures feel like bait. Caves stop being resource tunnels. Every distant sound has the manners of a threat.

The signature presence is The Anomaly: the thing that makes shelter feel temporary and confidence feel stupid.

WHY PLAY THIS PACK?

Cinematic dread
Fog, darkness, sound physics, visual damage, ambience, and hostile terrain make the world feel wrong fast.		 Audio matters
Footsteps, knocks, whispers, growls, echoes, and distant movement are warnings. Muting them is self-sabotage.
Stalking threats
Entities observe, ambush, mimic, chase, punish careless routes, and wait until you relax.		 Exploration with consequences
Structures, wilderness, caves, towns, and lonely paths offer loot, danger, and the occasional terrible idea.
Brutal nights
Night is not a timer. It is a pressure cooker where small mistakes become expensive.		 Survival over power fantasy
Gear helps. It does not delete fear. Ammo runs out. Panic ruins aim. Awareness stays king.

EXPERIENCE LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore forests, caves, structures, and cursed places for loot and answers.
  3. Listen before moving. Sound is often the first warning and the last mercy.
  4. Survive encounters you were absolutely not ready for.
  5. Return with better gear, worse nerves, and fewer lies about being safe.

BEFORE YOU INSTALL

  • Install from the official CurseForge project page or the CurseForge App.
  • Do not use random reupload sites. Malware loves impatient players.
  • Allocate at least 5GB RAM; use 6–8GB if your PC can spare it.
  • Use headphones for the intended sound-led horror experience.
  • Back up worlds before updates, server moves, or heavy config changes.
  • Extra mods, shaders, resource packs, and config edits are your responsibility. Test clean before reporting bugs.

INSTALLATION

  1. Install the CurseForge App.
  2. Open Minecraft inside CurseForge.
  3. Search for Still Watching.
  4. Confirm the project is by Soumyajit.
  5. Click Install.
  6. Allocate at least 5GB RAM.
  7. Launch the pack. Put on headphones. Regret your confidence.

Using another launcher? Use one that supports CurseForge modpacks properly, then install the latest compatible Forge 1.20.1 release from the official project page.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Do not dump client-only visual, shader, UI, map, or rendering mods into a dedicated server.
  • Test world generation and joining before inviting players.
  • Voice/audio setup is worth testing before the first real session.

A mismatched server pack is not mysterious horror. It is bad admin work wearing a cheap mask.

NEED HELP?

Before reporting an issue, reproduce it on a clean install if possible and include:

  • Modpack version
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Reports like “it broke” are haunted postcards. Bring evidence.

Report issues on GitHub or use the full installation guide.
