FORGE 1.20.1 HORROR SURVIVAL

STILL WATCHING

Fog hides movement. Sound becomes evidence. Darkness stops being empty.

A focused Minecraft horror survival modpack built around hostile exploration, stalking entities, brutal nights, and the feeling that something noticed you before you noticed it.

Current release: Still Watching V1.0.9. Update through CurseForge, back up your worlds, and keep multiplayer clients and servers on matching versions.

WHAT THIS PACK IS

Still Watching is not a kitchen-sink modpack with a few monsters duct-taped onto it. It is a horror survival experience where travel feels unsafe, structures feel suspicious, caves feel too deep, and every sound has permission to ruin your plan.

Build carefully. Explore slowly. Listen before you move. Gear helps, but awareness matters more.

CORE HORROR LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore forests, caves, structures, lonely paths, and cursed places.
  3. Listen for footsteps, echoes, movement, and sounds that do not belong.
  4. React when something turns your route home into a mistake.
  5. Return with better gear, worse nerves, and fewer lies about being safe.

KEY FEATURES

Fog-first atmosphere

Visibility is part of the threat. Distance, weather, terrain, and darkness turn normal travel into a decision with consequences.

Sound-led tension

Footsteps, knocks, whispers, growls, echoes, and distant movement become warnings before they become problems.

Stalking entities

Threats follow, ambush, mimic, punish careless routes, and wait for the exact moment you start acting comfortable.

Survival over power fantasy

This pack is about preparation, escape routes, resource pressure, and bad choices made under worse lighting.

EXPECT THIS

  • Fog-heavy horror exploration.
  • Sound-based tension and danger cues.
  • Stalking entities, hostile encounters, and moments where running is correct.
  • Forests, caves, structures, roads, cursed places, and risky loot routes.
  • Survival pressure without turning the game into a factory spreadsheet.
  • Multiplayer panic when everyone realizes the noise was not decorative.

DO NOT EXPECT THIS

  • A peaceful building therapy pack.
  • A random kitchen-sink profile with horror sprinkled on top.
  • A pure tech, factory, or automation experience.
  • A safe power fantasy where gear solves every problem.
  • A pack that stays balanced after adding random mods and hoping the launcher forgives you.

INSTALLATION

  1. Install the CurseForge App.
  2. Search for Still Watching.
  3. Confirm the project is by Soumyajit.
  4. Install or update to Still Watching V1.0.9.
  5. Allocate at least 5GB RAM; 6–8GB is preferred.
  6. Launch the pack. Use headphones. Lower the lights. Stop pretending you are calm.

Using another launcher? Use one that properly supports CurseForge modpacks and the correct Forge 1.20.1 setup.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Do not put client-only visual, shader, UI, map, or rendering mods into a dedicated server.
  • Test world generation and joining before inviting players.
  • Voice and audio setup is worth testing before the first real session.

A mismatched server pack is not mysterious horror. It is bad admin work wearing a cheap mask.

BEFORE REPORTING ISSUES

Reproduce the issue on a clean install if possible. If it still happens, report it with useful evidence.

  • Modpack version, for example Still Watching V1.0.9
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Report issues on GitHub or read the installation guide.
