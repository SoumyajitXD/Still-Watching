WHAT IS STILL WATCHING?

Still Watching is a cinematic Minecraft horror modpack designed to make survival feel dangerous again. This is not cozy survival with a few monsters sprinkled on top. This is a hostile world of fog-covered forests, abandoned structures, distorted sounds, brutal nights, and creatures that stalk, hunt, mimic, and wait.

The pack is built for atmosphere, fear, exploration, and pressure. It wants you cautious, underprepared, and listening to noises you would rather ignore.

WHY PLAY THIS PACK?

THE EXPERIENCE LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore structures, forests, caves, and cursed places for loot and answers.
  3. Listen for movement, knocks, whispers, and anything that sounds too close.
  4. Survive encounters you were absolutely not ready for.
  5. Return with better gear, worse nerves, and fewer illusions of safety.

BEFORE YOU INSTALL

  • Use the official CurseForge project page or the CurseForge App.
  • Do not use random reupload sites. Malware loves impatient players.
  • Make sure your PC can allocate at least 5GB RAM to Minecraft.
  • Use headphones if you want the intended horror experience.
  • Back up worlds before updates or heavy config changes.

INSTALLATION

  1. Install the CurseForge App.
  2. Open Minecraft inside CurseForge.
  3. Search for Still Watching.
  4. Confirm the project is by Soumyajit.
  5. Click Install.
  6. Allocate at least 5GB RAM.
  7. Launch the pack and use headphones.

NEED HELP?

Before reporting an issue, test on a clean install if possible and include:

  • Modpack version
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Reports like “it broke” are not bug reports. They are haunted postcards. Bring evidence.
