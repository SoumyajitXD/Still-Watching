STILL WATCHING

Forge 1.20.1 horror survival where fog hides movement, sound becomes evidence, and shelter is only a polite suggestion.

The fog is not empty. The silence is not safety.

Build carefully. Explore slowly. Listen before you move.

Official downloads belong on CurseForge. Use headphones, allocate enough RAM, and avoid random reupload sites unless you collect malware like trading cards.

Best played at night, with headphones, low brightness, and at least one terrible decision.

THE 10-SECOND PITCH

Still Watching turns Minecraft into a fog-choked horror survival run built around stalking entities, hostile exploration, oppressive audio, brutal nights, and the constant suspicion that something noticed you before you noticed it.

This is not a kitchen-sink modpack wearing a Halloween mask. It is focused, tense, and rude enough to punish players who sprint into the dark like horror movie extras with pickaxes.

EXPERIENCE LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore forests, caves, structures, and cursed places for loot and answers.
  3. Listen before moving. Sound is often the first warning and the last mercy.
  4. Survive encounters you were absolutely not ready for.
  5. Return with better gear, worse nerves, and fewer lies about being safe.

INSTALLATION

  1. Install the CurseForge App.
  2. Open Minecraft inside CurseForge.
  3. Search for Still Watching.
  4. Confirm the project is by Soumyajit.
  5. Click Install.
  6. Allocate at least 5GB RAM.
  7. Launch the pack. Put on headphones. Regret your confidence.

Using another launcher? Use one that supports CurseForge modpacks properly, then install the latest compatible Forge 1.20.1 release from the official project page.

BEFORE YOU INSTALL

  • Install from the official CurseForge project page or the CurseForge App.
  • Do not use random reupload sites. Malware loves impatient players.
  • Use headphones for the intended sound-led horror experience.
  • Back up worlds before updates, server moves, or heavy config changes.
  • Extra mods, shaders, resource packs, and config edits are your responsibility. Test clean before reporting bugs.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Do not dump client-only visual, shader, UI, map, or rendering mods into a dedicated server.
  • Test world generation and joining before inviting players.
  • Voice/audio setup is worth testing before the first real session.

A mismatched server pack is not mysterious horror. It is bad admin work wearing a cheap mask.
