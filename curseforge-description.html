STILL WATCHING

Forge 1.20.1 horror survival where fog hides movement, sound becomes evidence, and something in the dark notices you before you notice it.

The fog is not empty. The silence is not safety.

Build carefully. Explore slowly. Listen before you move.

Best played at night with headphones, low brightness, and just enough confidence to make your first mistake.

THE 10-SECOND PITCH

Still Watching is a focused Minecraft horror survival modpack built around fog, darkness, stalking entities, brutal nights, cursed exploration, and cinematic paranoia.

This is not a kitchen-sink pack wearing a cheap monster mask. The world is hostile by design: forests disappear into mist, structures feel like bait, caves stop being resource tunnels, and every distant sound has the manners of a threat.

Official downloads belong on CurseForge. Avoid random reupload sites unless malware is somehow part of your horror experience.

WHY THIS PACK HITS DIFFERENT

Fog-first atmosphere
Visibility is part of the threat. Distance, darkness, weather, and terrain turn normal travel into bad judgment with legs.		 Sound-led horror
Footsteps, knocks, whispers, growls, echoes, and distant movement are warnings, traps, and occasionally your obituary warming up.
Stalking entities
Threats do not exist only to jumpscare you once and vanish. They follow, ambush, mimic, punish careless routes, and wait for you to relax.		 Survival over power fantasy
Gear helps, but it does not make you untouchable. Awareness, preparation, escape routes, and not being stupid matter more.

THE EXPERIENCE LOOP

  1. Prepare before nightfall. Panic is expensive.
  2. Explore forests, caves, structures, lonely paths, and cursed places for loot and answers.
  3. Listen before moving. Sound is often the first warning and the last mercy.
  4. React when stalking entities turn your plan into comedy.
  5. Return with better gear, worse nerves, and fewer lies about being safe.

EXPECT THIS

  • Fog-heavy horror exploration.
  • Sound-based tension and danger cues.
  • Stalking entities, hostile encounters, and moments where running is the correct strategy.

MULTIPLAYER / SERVER NOTES

  • Use the same modpack version on server and clients.
  • Match Minecraft 1.20.1 and Forge versions exactly.
  • Keep backups before updates or server migrations.
  • Do not dump client-only visual, shader, UI, map, or rendering mods into a dedicated server.
  • Test world generation and joining before inviting players.
  • Voice/audio setup is worth testing before the first real session.

A mismatched server pack is not mysterious horror. It is bad admin work wearing a cheap mask.

BEFORE REPORTING ISSUES

Reproduce the issue on a clean install if possible. If it still happens, report it with useful evidence, not a haunted postcard that says “it broke.”

  • Modpack version
  • Launcher used
  • Minecraft version
  • Forge version, if visible
  • Java version, if relevant
  • Singleplayer or multiplayer
  • Crash report or latest.log
  • Steps to reproduce the problem
  • Any extra mods, shaders, resource packs, or config edits you added yourself

Report issues on GitHub or read the installation guide.
