## Before You Install

Make sure you have:

1. A valid Minecraft: Java Edition account.
2. The CurseForge App, or another launcher that supports CurseForge modpacks.
3. Enough system RAM to allocate at least `5 GB` to the pack.
4. Headphones.

Still Watching is built around fog, darkness, sound, stalking entities, and pressure. Playing with weak audio or daylight-soup video settings is technically allowed, but spiritually embarrassing.

---

## Recommended Method: CurseForge App

This is the easiest and safest install path.

1. Download and install the **CurseForge App**.
2. Open the app.
3. Go to **Minecraft**.
4. Open **Browse** or **Discover**.
5. Search for **Still Watching**.
6. Confirm the project author is **Soumyajit**.
7. Click **Install**.
8. Wait for the app to download the correct files.
9. Open **My Modpacks**.
10. Select **Still Watching**.
11. Allocate at least `5 GB` RAM.
12. Click **Play**.

The CurseForge App handles the modpack profile and dependencies for you. Do not manually drag random `.jar` files into the profile unless you enjoy converting bugs into modern art.

## Installing with Another Compatible Launcher

Use this only if your launcher supports CurseForge modpacks properly.

General steps:

1. Open your launcher.
2. Choose the option to import or install a CurseForge modpack.
3. Search for **Still Watching**, or use the CurseForge project page.
4. Install the latest release for:
   - Minecraft: `1.20.1`
   - Loader: `Forge`
5. Allocate at least `5 GB` RAM.
6. Launch the pack.

If your launcher asks for a downloaded modpack archive, download it from the official CurseForge page. Do not use random reuploads. That is how you invite malware to dinner and then act surprised when it eats the house.

## RAM Allocation

Recommended allocation: `5 GB` or higher.

Suggested range:

| System RAM | Suggested Allocation |
| --- | --- |
| `8 GB` | `5 GB` if the system can spare it |
| `12 GB` | `5 GB` to `6 GB` |
| `16 GB+` | `6 GB` to `8 GB` |

Do not allocate all your system RAM to Minecraft. Your operating system still needs memory. Starving the OS to feed Minecraft is not optimization; it is a hostage situation.

## Multiplayer / Server Notes

Still Watching supports multiplayer, but server setup depends on how the pack is distributed and what server files are available for the current version.

For a dedicated server:

1. Use the same modpack version on both server and clients.
2. Use Minecraft `1.20.1`.
3. Use the matching Forge server version required by the pack.
4. Keep server-side configs aligned with the client pack.
5. Do not install client-only visual/audio/performance mods on the server.
6. Check `latest.log` and crash reports when something breaks.

When reporting multiplayer issues, always say whether the problem happens in:

- Singleplayer only
- Multiplayer only
- Both singleplayer and multiplayer
- Dedicated server, LAN, or hosted server

Logs are not decoration. They are evidence. Without them, debugging becomes séance work.

## Updating the Modpack

Recommended update method:

1. Open the CurseForge App.
2. Go to **My Modpacks**.
3. Select **Still Watching**.
4. Check for updates.
5. Update through the launcher.
6. Back up worlds before launching the updated version.

Before updating an important world:

- Back up your saves.
- Back up any custom configs.
- Remove extra mods you added yourself if troubleshooting.
- Read the file/version notes when available.

Never test a modpack update on your only copy of a world. That is not bravery. That is speedrunning regret.

## Troubleshooting

### The pack does not launch

Try this:

1. Restart the launcher.
2. Make sure the pack is installed as a Forge `1.20.1` profile.
3. Allocate at least `5 GB` RAM.
4. Remove extra mods you added manually.
5. Repair or reinstall the profile from CurseForge.
6. Check the crash report or `latest.log`.

### The game crashes while loading

Common causes:

- Not enough RAM allocated.
- Wrong Minecraft version.
- Wrong loader.
- Broken manual mod additions.
- Corrupted profile install.
- Outdated graphics drivers.

### The game is laggy

Try this:

1. Lower render distance.
2. Lower simulation distance.
3. Avoid stacking extra shaders or resource packs.
4. Close background apps.
5. Keep the recommended performance mods included with the pack intact.
6. Do not turn every visual setting into a bonfire and then ask why the frames died.

### Audio feels wrong or too quiet

Still Watching uses sound as part of the horror design.

Try this:

1. Use headphones.
2. Keep hostile creature and ambient audio audible.
3. Check Minecraft audio sliders.
4. Check system audio output.
5. Disable external audio enhancements if they mangle positional sound.

### Multiplayer connection issues

Check:

1. Client and server are on the same modpack version.
2. Client and server use the same Minecraft/Forge target.
3. The server does not contain client-only mods.
4. The server has enough RAM.
5. The server log does not show missing mods or config mismatches.
